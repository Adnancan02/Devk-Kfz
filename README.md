# DEVK Test Automation Project
Java, Playwright, Cucumber and Allure based UI test automation example for the DEVK KFZ tariff calculator flow.

## Tech Stack

- Java 21
- Gradle
- Playwright Java
- Cucumber JVM with JUnit 4
- Allure Cucumber adapter

## Project Structure

```text
app/src/test/java/com/devk/test/base      Cucumber hooks and browser lifecycle
app/src/test/java/com/devk/test/config    Runtime configuration
app/src/test/java/com/devk/test/pages     Page object layer
app/src/test/java/com/devk/test/steps     Cucumber step definitions
app/src/test/resources/features           Gherkin feature files
```

## Run Tests

```bash
./gradlew test
```

On Windows:

```powershell
.\gradlew.bat test
```

Useful runtime properties:

```bash
./gradlew test -Dbrowser=chromium -Dheadless=true -DbaseUrl=https://www.devk.de/ -Dcucumber.filter.tags=@smoke
```

Supported browser values:

- `chromium`
- `chrome`
- `edge`
- `firefox`
- `webkit`
- `safari`

The same configuration can also be passed through environment variables:

```text
TEST_BROWSER=chromium
TEST_HEADLESS=true
TEST_BASE_URL=https://www.devk.de/
TEST_TIMEOUT=30000
TEST_KEEP_VIDEOS=false
```

## Reports and Artifacts

Generated reports are written under `app/build`:

- Cucumber HTML: `app/build/reports/cucumber.html`
- Cucumber JSON: `app/build/reports/cucumber.json`
- Screenshots: `app/build/reports/screenshots`
- Videos: `app/build/videos`

Allure result files are generated by the Cucumber plugin and ignored by Git.

## Design Notes

The framework uses Cucumber for business-readable scenarios and 
Playwright for reliable browser automation, including Shadow DOM support. 
Browser configuration is centralized in `TestConfig`, while `Hooks` owns 
Playwright lifecycle and per-scenario browser contexts.

The test intentionally covers one end-to-end happy path as an interview example. 
For a production suite, the next improvements would be broader test data management, 
mocked/stable environments for volatile third-party content, 
and additional negative and boundary scenarios
